# Instructions
* Required libraries: 
    * glad 
    * glfw 
    * glm 
    * freetype
* To run the code:
    ```
    mkdir build
    cd build
    cmake ..
    make
    ./window
    ```
# Controls
|          Effect        |   Keyboard input    |          
|:----------------------:|:-------------------:|
| Move player up         |          W          |
| Move player left       |          A          |
| Make player down       |          S          |
| Make player right      |          D          |
| Exit                   |         Esc         |
